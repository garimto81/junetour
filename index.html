🚀 왕초보를 위한 AI 웹페이지 제작 가이드

AI로 만든 결과물을 실제 웹사이트로 만드는 모든 과정을 따라 해보세요!

💻 시작 전 준비물

  • 인터넷에 연결된 컴퓨터
  • GitHub (깃허브) 계정 (없다면 지금 만들어 보세요!)

1단계: AI에게 결과물 요청하기

먼저, 웹페이지에 담을 콘텐츠를 AI(GPT, Claude, Gemini 등)에게 요청합니다. 여기서는 예시로 "세계 5대 추천 여행지" 목록을 만들어 보겠습니다.

(AI에게 보낼 메시지 예시)

세계에서 가장 추천하는 여행지 5곳을 순서대로 알려줘. 각 여행지에 대한 한 줄 설명도 포함해줘.

2단계: AI에게 웹페이지용 코드로 변환 요청하기

AI가 만들어준 텍스트를 웹페이지의 '설계도(코드)'로 바꿔달라고 요청할 차례입니다. 아래 프롬프트를 그대로 사용하는 것이 핵심입니다!

(AI에게 보낼 메시지 예시)

방금 네가 만들어준 여행지 목록을 가지고 하나의 HTML 파일로 웹페이지를 만들어줘. 초보자도 쓰기 쉽게 CSS 스타일과 모든 내용을 HTML 파일 안에 전부 포함시켜줘. 디자인은 깔끔하고 보기 좋게 부탁해!

그러면 AI가 아래와 같은 HTML 코드를 생성해 줄 것입니다.

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>세계 5대 추천 여행지</title>
    <style>
        /* 여기에 CSS 스타일 코드가 들어갑니다 */
    </style>
</head>
<body>
    <!-- 여기에 콘텐츠 내용이 들어갑니다 -->
</body>
</html>

3단계: GitHub에 코드 올려 웹페이지 만들기

이제 여러분의 코드를 인터넷 세상에 공개할 시간입니다. GitHub이라는 코드 저장소에 우리만의 프로젝트 공간을 만들고, 그곳에 AI가 만들어준 코드를 올려보겠습니다.

1. 새 저장소(Repository) 만들기

GitHub에 로그인 후, 'Create new repository' 버튼을 클릭하세요. 저장소 이름을 my-awesome-trip 처럼 정하고, 'Public' 및 'Add a README file'을 체크한 뒤 생성합니다.

2. 새 파일(index.html) 생성하기

만들어진 저장소에서 'Add file' > 'Create new file'을 선택하세요. 파일 이름은 반드시 index.html이라고 입력해야 합니다.

3. 코드 붙여넣고 저장하기

2단계에서 AI가 만들어 준 HTML 코드 전체를 복사해서 큰 편집창에 붙여넣습니다. 그리고 페이지 맨 아래의 'Commit new file' 버튼을 클릭하여 저장하세요.

4단계: GitHub Pages로 진짜 웹페이지 발행하기

마지막 단계입니다! 업로드한 코드를 실제 인터넷 주소로 바꾸어 누구나 접속할 수 있게 만듭니다.

1. Pages 설정으로 이동하기

프로젝트 저장소 상단의 'Settings' 탭을 누른 후, 왼쪽 메뉴에서 'Pages'를 클릭하세요.

2. 웹페이지 주소 활성화하기

'Branch' 섹션에서 드롭다운 메뉴를 'None'에서 main으로 바꾸고 'Save'를 누르세요. 1~2분 뒤 페이지가 새로고침되면, 상단에 여러분의 웹사이트 주소가 나타납니다!

🎉 축하합니다! 이제 여러분은 어떤 아이디어든 웹페이지로 만들 수 있습니다! 🎉
