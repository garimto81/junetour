🚀 왕초보를 위한 AI 웹페이지 제작 가이드

AI로 만든 결과물을 실제 웹사이트로 만드는 모든 과정을 따라 해보세요!

💻 시작 전 준비물

  • 인터넷에 연결된 컴퓨터
  • GitHub (깃허브) 계정 (없다면 지금 만들어 보세요!)

1단계: AI에게 결과물 요청하기

먼저, 웹페이지에 담을 콘텐츠를 AI(GPT, Claude, Gemini 등)에게 요청합니다. 여기서는 예시로 "세계 5대 추천 여행지" 목록을 만들어 보겠습니다.

(AI에게 보낼 메시지 예시)

세계에서 가장 추천하는 여행지 5곳을 순서대로 알려줘. 각 여행지에 대한 한 줄 설명도 포함해줘.

2단계: AI에게 웹페이지용 코드로 변환 요청하기

AI가 만들어준 텍스트를 웹페이지의 '설계도(코드)'로 바꿔달라고 요청할 차례입니다. 아래 프롬프트를 그대로 사용하는 것이 핵심입니다!

(AI에게 보낼 메시지 예시)

방금 네가 만들어준 여행지 목록을 가지고 하나의 HTML 파일로 웹페이지를 만들어줘. 초보자도 쓰기 쉽게 CSS 스타일과 모든 내용을 HTML 파일 안에 전부 포함시켜줘. 디자인은 깔끔하고 보기 좋게 부탁해!

그러면 AI가 아래와 같은 HTML 코드를 생성해 줄 것입니다.

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>세계 5대 추천 여행지</title>
    <style>
        /* 여기에 CSS 스타일 코드가 들어갑니다 */
    </style>
</head>
<body>
    <!-- 여기에 콘텐츠 내용이 들어갑니다 -->
</body>
</html>

3단계: 내 컴퓨터에서 미리보기 (확인)

코드가 웹페이지처럼 잘 보이는지 내 컴퓨터에서 먼저 확인합니다.

  1. 컴퓨터의 기본 메모장 또는 TextEdit을 켭니다.
  2. AI가 준 코드 전체를 복사하여 붙여넣습니다.
  3. 파일을 저장할 때, 파일 이름을 index.html 로, 파일 형식을 '모든 파일(All files)'로 지정하여 저장합니다.
  4. 저장된 `index.html` 파일을 더블클릭하면 웹 브라우저에서 결과물을 미리 볼 수 있습니다.

4단계: GitHub에 코드 올리기

이제 코드를 인터넷 공간(GitHub)에 올려서 다른 사람들도 볼 수 있게 준비합니다.

  1. GitHub에 로그인 후, 'Create new repository'를 클릭합니다.
  2. 저장소 이름(Repository name)을 my-first-webpage 등으로 입력합니다. (영문, 공백 없음)
  3. 'Public'을 반드시 선택하고, 'Add a README file'에 체크합니다.
  4. 'Create repository' 버튼을 눌러 저장소를 만듭니다.
  5. 만들어진 저장소에서 'Add file' > 'Upload files'를 클릭합니다.
  6. 아까 만든 index.html 파일을 업로드하고, 'Commit changes' 버튼을 눌러 저장합니다.

5단계: GitHub Pages로 진짜 웹페이지 발행하기

마지막 단계입니다! 업로드한 코드를 실제 인터넷 주소로 만듭니다.

  1. 프로젝트 저장소 상단의 'Settings' 탭을 클릭합니다.
  2. 왼쪽 메뉴에서 'Pages'를 클릭합니다.
  3. 'Branch' 섹션에서 드롭다운 메뉴를 'None'에서 main으로 변경하고 'Save'를 누릅니다.
  4. 잠시 (1~2분) 기다리면, 페이지 상단에 "Your site is live at..." 라는 메시지와 함께 웹사이트 주소가 나타납니다.
  5. 그 주소를 클릭하면, 여러분만의 웹페이지가 완성됩니다! 이 주소는 누구에게나 공유할 수 있습니다.

🎉 축하합니다! 이제 여러분은 어떤 아이디어든 웹페이지로 만들 수 있습니다! 🎉
